Deploy React Project with Firebase

Step by step how to.

Posted by aquasar on Dec 1st, 2018

What is Firebase?

Firebase is an easy to use baas (backend as as service) for app and web developers. Firebase (aquired by Google) integrates with Google Cloud platform (Google's cloud computing services and direct competitor to AWS and Microsoft Azure). It provides a good number of features including a realtime database, syncing data automatically through a websocket.

Other essential features it provides include authentication,storage and hostings. This makes firebase a one-stop shop for creating and deploying a starter or simple full stack web app.

Firebase is free to use for small apps and the pricing grows modestly afterward ( see pricing )

Create production build in react

In your finished react app, first make sure to create a `production` build of your react project. In the terminal of your project folder:

   
            npm run build


Install firebase tools

In the terminal inside your project folder:

   
            npm install -g firebase-tools


Deploy

In the terminal inside your project folder: 

   
              firebase login
              firebase init

  • Inside prompt use BUILD as the public directory.
  • Configure as single page app YES
  • Overwrite build/index.html file NO
  • Next run firebase deploy. This should give you a temp link to your site which you can share with others.
